数列极限的定义

数列：α1,α2,…,αn,…\alpha_{1},\alpha_{2},\dots,\alpha_{n},\dotsα1​,α2​,…,αn​,…这样排成一列的实数称为数列。

如果 ana_{n}an​ 随着 nnn 充分增大，逐渐接近为一个实数 α\alphaα 时，称，数列 {αn}\{\alpha_{n}\}{αn​} 收敛于α\alphaα。

但是，“充分增大”，“逐渐接近”这些词是模糊不清，缺乏严谨的，应该避免使用。

很容易知道，如果 {αn}\{\alpha_{n}\}{αn​} 收敛于 α\alphaα，说明，随着 nnn 的增大，αn\alpha_{n}αn​ 与 α\alphaα 的距离会越来越小，即，无论我们取多么小的一个距离 ε\varepsilonε，随着 nnn 的增大，某一项 n0(ε)n_{0}(\varepsilon)n0​(ε) （可以由 ε\varepsilonε 决定，是一个关于 ε\varepsilonε 的表达式）之后，αn\alpha_{n}αn​ 与 α\alphaα，之间的距离总会比这个距离还小。

这样的表述很好的避免使用含糊不清的词，仅用有限的正整数和实数表达出来了极限的概念。

极限与极限点

Definition: 数列的极限
设{αn}\{\alpha_{n}\}{αn​}为一个数列，给定任意正实数 ε\varepsilonε，当 n>n0(ε)n>n_{0}(\varepsilon)n>n0​(ε) 时，∣αn−α∣<ε|\alpha_{n}-\alpha|<\varepsilon∣αn​−α∣<ε总是成立，那么称数列{αn}\{\alpha_{n}\}{αn​}收敛，数列的极限为 lim⁡n→∞αn=α\lim_{ n \to \infty }\alpha_{n}=\alphalimn→∞​αn​=α。

为什么要用 n0(ε)n_{0}(\varepsilon)n0​(ε) 而不是 NNN：主要是为了显式体现出 ε\varepsilonε 与其的关系，某些地方省略也不影响时也会使用 NNN 或直接 n0n_{0}n0​

Definition: 数列的聚点
在某点附近无论多么小的范围内，都有无限个项，数列{αn}\{\alpha_{n}\}{αn​}，给定任意正实数ε\varepsilonε， 如果满足 ∣αn−β∣<ε|\alpha_{n}-\beta|<\varepsilon∣αn​−β∣<ε 的项有无穷个，则 β\betaβ 是 {αn}\{\alpha_{n}\}{αn​} 的一个聚点，如果数列存在极限，则聚点有且只有一个，为该数列的极限（证明见下文）。

与极限的关系：聚点只要求有无穷个项在该点附近，一个数列可能会有多个极限点。
